canvasの使い方

canvasを作る

canvasTest.htmlとcanvasTest.cssというファイルを作成し、 それぞれに次のコードを入力してください。 


    <!DOCTYPE html>
    <html lang="jn">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="styleSheet" href="canvasTest.css">
        <title>canvasのテスト</title>
    </head>
    <body>
        <h1>Canvasのテスト<h1>
        <canvas width="768px" height="512px" id="canvas"></canvas>
    </body>
    </html>
                


    canvas{
        border:1px solid #000000;
    }

成功すると、画面にcanvasが表示されるはずです。
widthで横幅、heightで縦幅を変更できます。
cssはcanvasが視覚的に見えるようにするための処理であるため、 border:1px solid #000000;の部分を background-color:black;などに変えても構いません。

canvasに四角形を描画する

htmlのbodyの最下部に次のようなコードを追加し、 canvasTest.jsというファイルを作成してこのようなコードを入力してください。 


    <script type="text/JavaScript" src="canvasTest.js"></script>
                


    const canvas=document.getElementById("canvas");
    const ctx=canvas.getContext("2d");

    ctx.beginPath();
    ctx.rect(100,100,50,50);
    ctx.fillStyle="#000000";
    ctx.fill();
    ctx.closePath();

canvasに四角形が描画出来たら成功です。
document.getElementById("canvas");はcanvasというidを持つhtmlの要素を取得するプログラムであり、 ctx.beginPath();からctx.closePath();の間で四角形を描画しています。
ctx.rectは第一引数から順に(四角形の左端のx座標,四角形の上端のy座標,四角形の横幅,四角形の縦幅)を入力します。
ctx.fillStyleは四角形の色を選択でき、ctx.fill()で四角形の描画が実行されます。
ctx.fillStyleをctx.strokeStyle、 ctx.fill()をctx.stroke()に変更すると、 枠線を描画することができます。ctx.lineWidth=数字で、枠線の幅を調整できます。
fillとstrokeは同時に処理することができ、枠線と中身の色を変えられます。

canvasに四角形が描画されている様子

canvasに文字を描画する

jsのctx.beginPath();からctx.closePath();の コードを次のように変更してください。 


    ctx.font="48px serif";
    ctx.fillStyle="#000000";
    ctx.fillText("Hello World",10,50);

canvasにHello Worldという文字列が描画できたら成功です。
ctx.fontは文字のフォントを設定するコードで、 他に"32px Arial"なども設定できます。
ctx.fillStyleは文字の色を選択できます。 ctx.fillTextで文字を入力でき、 このとき第一引数から順に(入力する文字,文字のx座標の左端,文字のy座標の下端)と入力します。
四角形と違い、y座標の下端を入力しなければならない点に注意してください。
文字も四角形と同じようにctx.fillStyleをctx.strokeStyle、 ctx.fillTextをctx.strokeTestとすることで 文字の枠線が入力できます。
